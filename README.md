# Sistem za upravljanje školskom bibliotekom

Ovaj repozitorijum sadrži projekat sistema za upravljanje bazom podataka školske biblioteke.

## Osnovne informacije
* **Tehnologija:** Microsoft SQL Server (T-SQL)
* **Tip projekta:** Školski projekat

## Dokumentacija
Sve detaljne informacije o strukturi baze podataka (3NF), implementiranoj poslovnoj logici (stored procedure, trigeri, funkcije, pogledi), kao i strategiji optimizacije i indeksiranja, nalaze se u priloženoj dokumentaciji u okviru ovog repozitorijuma.
